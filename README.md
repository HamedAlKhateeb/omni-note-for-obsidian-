# OmniNote for Obsidian (إضافة أومني نوت)

إضافة متكاملة (Sidebar Plugin) لمستخدمي أوبسيديان (Obsidian) باللغة العربية، تجمع بين أهم أدوات الإنتاجية في شريط جانبي واحد ومريح، مع تصميم عصري وبسيط.

## 🌟 الميزات الأساسية (Features)

1. **💡 حكمة اليوم (Quote of the Day):**
   - عرض حكم واقتباسات ملهمة متجددة.
   - إمكانية تكبير وتصغير الخط بسهولة.
   - إمكانية استيراد حكم جديدة من ملف `CSV`.

2. **⏱️ بومودورو (Pomodoro Timer):**
   - مؤقت متكامل لجلسات العمل وفترات الاستراحة.
   - إمكانية تعديل مدة الجلسة والاستراحة مباشرة من الواجهة.
   - **جديد:** زر **تخطي (Skip)** للانتقال السريع والذكي بين فترات العمل والاستراحة.
   - عرض حالة المؤقت (عمل / استراحة) بلون متوافق مع ثيم أوبسيديان (Accent Color).
   - توثيق وتسجيل جلسات البومودورو المكتملة وربطها بالملاحظة النشطة حالياً.

3. **📅 التقويم والمهام (Calendar & Tasks):**
   - تقويم شهري بتصميم نظيف يوضح الأيام الحالية.
   - إمكانية اختيار أي يوم وإضافة "ملاحظة اليوم" (Sticky Note).
   - جدول زمني لإضافة مهام ومواعيد محددة (بتوقيت معين) داخل كل يوم، مع إمكانية حذف المهام المنجزة.
   - شارات (Badges) توضح عدد المهام المسجلة لكل يوم على التقويم نفسه.

4. **📊 متابعة الإنجاز (Progress Tracker):**
   - إضافة مهام أو أهداف طويلة المدى وتتبع نسبة الإنجاز بوحدات مخصصة (صفحة، دقيقة، درس...).
   - عرض شريط تقدم مرئي لكل هدف لتوضيح مستواك الحالي.

## 🚀 التثبيت وطريقة الاستخدام

1. انسخ ملفات الإضافة (`main.js`، `manifest.json`، `styles.css`) إلى مجلد الإضافات في قبو أوبسيديان الخاص بك:
   `YourVault/.obsidian/plugins/omni-note-extension/`
2. اذهب إلى الإعدادات في أوبسيديان (`Settings` -> `Community Plugins`).
3. قم بتفعيل إضافة `OmniNote Extension`.
4. افتح الشريط الجانبي الأيمن، وستجد أيقونة `OmniNote` (📅/⏱️). انقر عليها لفتح لوحة التحكم الخاصة بك.
