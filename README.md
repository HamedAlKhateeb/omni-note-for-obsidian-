# 📝 OmniNote Extension

إضافة **OmniNote** هي إضافة متكاملة (Plugin) لبرنامج [Obsidian](https://obsidian.md/)، تهدف إلى تعزيز إنتاجيتك وتنظيم وقتك وملاحظاتك من خلال شريط جانبي مدمج (Sidebar) يوفر مجموعة من الأدوات الأساسية في مكان واحد.

## ✨ المميزات (Features)

*   🍅 **مؤقت بومودورو (Pomodoro Timer):** إدارة وقتك بفعالية وزيادة التركيز أثناء العمل أو الدراسة داخل أوبسيديان مباشرة.
*   📅 **تقويم متقدم (Calendar):** تقويم مدمج مع إمكانية إضافة "ملاحظات لاصقة" (Sticky notes) لترتيب مواعيدك ومهامك اليومية بسهولة.
*   💡 **إشعارات الاقتباسات (Quote Notifications):** احصل على إشعارات دورية لاقتباسات ملهمة أو ملاحظات هامة للحفاظ على حافزك.
*   📱 **شريط جانبي مدمج (Integrated Sidebar):** واجهة مستخدم نظيفة ومنظمة تتكامل بشكل مثالي مع واجهة أوبسيديان، مما يسهل الوصول لجميع هذه الأدوات دون تشتت.

## 🚀 التثبيت (Installation)

بما أن الإضافة في مرحلة التطوير (أو للتثبيت اليدوي)، يمكنك اتباع الخطوات التالية:
1. قم بتحميل ملفات الإضافة (`main.js`, `manifest.json`, `styles.css`).
2. اذهب إلى مجلد التخزين (Vault) الخاص بك في أوبسيديان.
3. افتح مسار الإضافات: `.obsidian/plugins/`
4. قم بإنشاء مجلد جديد باسم `omni-note-extension`.
5. انسخ الملفات الثلاثة داخل هذا المجلد.
6. أعد تشغيل أوبسيديان، واذهب إلى الإعدادات > الإضافات المجتمعية (Community Plugins)، وقم بتفعيل **OmniNote Extension**.
